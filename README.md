# Rockethelp

The product of a week-long event hosted by #rocketseat with the objective of teaching how to create a sort of "TODO list" mobile app for IT tech inside a company.

## Techs Used
- React Native
- Typescript
- Expo
- Firebase
- React Navigation
- Native Base
